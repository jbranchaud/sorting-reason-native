# sorting

> Implementations of some sorting algorithms in ReasonML

This is a tiny command line program written in [ReasonML](https://reasonml.github.io/) and compiled natively using `bsb-native`.

### Featured

- [x] Insertion Sort
- [ ] Selection Sort
- [ ] Merge Sort
- [ ] Quick Sort

### Install

```
$ npm install
```

### Build

```
$ npm run build
```

### Run

```
$ lib/bs/bytecode/sorting
```
